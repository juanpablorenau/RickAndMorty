# RickAndMorty

An android app built using Kotlin that consumes Rick and Morty API to display characters,episodes,Location from the TV Series. It has been built following:

- Clean Architecture Principle
- Repository Pattern
- MVVM
- Flows
- Courrutines
- Room
- Retrofit
- Testing.
